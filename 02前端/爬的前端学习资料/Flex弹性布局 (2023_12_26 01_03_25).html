一、弹性布局的基本概念

Flexible Box模型，通常被称为flexbox，是一种一维的布局模型，也就是我们说的弹性盒子布局。它可以提供一个简便有效的、响应式的实现各种页面布局的方式，并且它给flexbox的子元素之间提供了强大的空间分布和对齐能力。弹性布局的核心思想是将父容器转变成弹性盒子，让父容器能够改变它子元素的宽度、高度和顺序，以最合适的方式填充可用的空间。


传统的布局方案基于盒子模型，它依赖display属性+position属性+float属性，对于一些特殊的布局实现非常不方便，比如将未知宽高的元素进行水平垂直居中。flexbox实现这种布局就非常容易，flexbox默认会存在两个轴：水平轴 和 垂直轴，它的子元素会基于这两条轴进行布局。


设定为flexbox后，其内部的子元素的float、clear、vertical-align属性将失效


采用Flex布局的元素，会变成弹性盒子，我们叫他“Flex容器”，它所有的子元素作为它的内部成员，他们被称作“项目”。


Flex容器默认存在两根轴：水平的主轴(main axis)和垂直的交叉轴(cross axis) ，主轴的开始位置叫做main start，主轴的结束位置叫做main end；交叉轴的开始位置叫cross start，交叉轴的结束位置叫cross end。容器的内部成员默认沿主轴的main start到main end方向排列。


二、创建弹性布局

想要使用flex布局，需要给元素添加display属性，将display属性的值设置为flex或者inline-flex。

  1. flex会生成一个块级的flex容器
  2. inline-flex会生成一个行内的flex容器
<style>
.page-wrap{
display:flex | inline-flex;
}
</style>
<div class="page-wrap">
<div class="box"></div>
</div>


三、父容器属性

（1）flex-direction

flex-direction属性定义了主轴的方向，那么连带着规定了flex容器内部成员的排列方向。主轴方向默认的是水平方向，主轴起点和主轴终点与内容方向相同。

flex-direction属性有4个值可以定义主轴的方向

（Ⅰ）flex-direction:row 主轴沿水平方向，起点到终点的方向是从左到右

（Ⅱ）flex-direction:row-reverse 主轴沿水平方向，起点到终点的方向是从右到左

（Ⅲ）flex-direction:column 主轴沿垂直方向，起点到终点的方向是从上到下

（Ⅳ）flex-direction:column-reverse 主轴沿垂直方向，起点到终点的方向是从下到上


（2）flex-wrap

flex-wrap属性指定容器内部成员是单行排列还是多行排列，如果允许换行排列，该属性该可以控制行的堆叠方向。它可以接受的值有3个，分别是：nowrap、wrap、wrap-reverse

（Ⅰ）flex-wrap:nowrap

指定容器内部成员单行排列，不允许换行。如果主轴空间固定并且空间不足时，会自动对内部成员进行调整，即此时内部成员沿着主轴方向的大小会进行调整。

如在水平方向上，内部成员width属性值为200，容器width属性值为600，不允许换行，则内部成员实际宽度会进行适配调整

（Ⅱ）flex-wrap:wrap

允许内部成员换行显示，如果主轴空间固定并且空间不足时，会自动换行。换行后的每一行的显示位置是从交叉轴起点到终点方向，即第一行到最后一行的排列是沿着交叉轴的起点位置到交叉轴的终点位置。

（Ⅲ）flex-wrap:wrap-reverse

允许内部成员换行显示，如果主轴空间固定并且空间不足时，会自动换行。换行后的每一行的显示位置是从交叉轴终点到起点方向，即第一行到最后一行的排列是沿着交叉轴的终点位置到交叉轴的起点位置

如下图案例，就是第一行在下，第二行在上


（3）flex-flow

flex-flow属性是flex-direction和flex-wrap的简写，默认是flex-flow:row nowrap。是CSS代码优化的一种技巧

.box{
flex-flow:row-reverse wrap;
}

（4）justify-content

justify-content属性规定如何分配弹性容器主轴的元素间及其周围的空间，即规定了内部成员的在主轴上的对齐方式。

弹性容器相关的属性值有5ge

.container{
justify-content:flex-start | flex-end | center | space-between | space-around;
}

注意：justify-content属性并不只是作用域弹性容器，还作用于网格容器，所以它的值并不只以上5个


（Ⅰ）flex-start

沿着主轴起点到终点方向对齐，后续的弹性元素与前一个对齐，以flex-direction:row为例是左对齐

（Ⅱ）flex-end

沿着主轴终点到起点方向对齐，其他的元素与后一个对齐，以flex-direction:row为例是右对齐

（Ⅲ）center

沿着主轴中点排列对齐，每行的第一个元素到行首的距离与每行最后一个元素到行尾的距离相同

（Ⅳ）space-between

在每行上均匀分配弹性元素，将领元素间距离相同。每行第一个元素与行首对齐，每行最后一个元素与行尾对齐

（Ⅴ）space-around

在每行上均匀分配弹性元素。相邻元素间距离相同。每行第一个元素到行首的距离和每行最后一个元素到行尾的距离将会是相邻元素之间距离的一半。


（5）align-items

align-items属性规定容器成员在交叉轴上的对齐方式，与弹性布局相关的属性值有5个

.container{
justify-content:flex-start | flex-end | center | baseline | stretch;
}

注意：align-items属性并不只是作用域弹性容器，还作用于网格容器，所以它的值并不只以上5个

该属性和justify-content属性一样都是依赖于容器的轴的位置，为了方便，这里都以水平方向为例，即flex-direction属性的值为row。

（Ⅰ）flex-start

元素向交叉轴起点对齐

（Ⅱ）flex-end

元素向交叉轴终点对齐

（Ⅲ）center

元素在交叉轴方向上居中

（Ⅳ）baseline

后续项目于项目的第一行文字的基线对齐

（Ⅴ）stretch

如果项目为设置高度或高度为auto，则项目自动占满容器的高度


（6）align-content

align-content属性定义多跟轴线的对齐方式，如果项目著有一根轴线，则该属性不起作用。也就是要使用该属性，必须要将flex-wrap属性的值设置为非nowrap。

（Ⅰ）flex-start

所有行从垂直轴起点开始填充。第一行的垂直轴起点边和容器的垂直轴起点边对齐。接下来的每一行紧跟前一行。

（Ⅱ）flex-end

所有行从垂直轴末尾开始填充。最后一行的垂直轴终点和容器的垂直轴终点对齐。同时所有后续行与前一个对齐。

（Ⅲ）center

所有行朝向容器的中心填充。每行互相紧挨，相对于容器居中对齐。容器的垂直轴起点边和第一行的距离相等于容器的垂直轴终点边和最后一行的距离。

（Ⅳ）space-between

所有行在容器中平均分布。相邻两行间距相等。容器的垂直轴起点边和终点边分别与第一行和最后一行的边对齐。

（Ⅴ）space-around

所有行在容器中平均分布，相邻两行间距相等。容器的垂直轴起点边和终点边分别与第一行和最后一行的距离是相邻两行间距的一半。

（Ⅵ）stretch

拉伸所有行来填满剩余空间


四、子成员属性

（1）flex-grow

flex-grow规定了项目在容器中占据剩余空间的相对比例，该属性也依赖于主轴方向。水平方向上分配的是宽度；垂直方向上分配的是高度。

该属性的值是一个系数，默认是0，代表即使存在剩余空间，也不进行缩放调整。如果所有项目的值相同，则对剩余空间进行等分。


此处盒子中的数字代表flex-grow的值


如果其中一个项目的值是其它项目值的两倍，那么占据的剩余空间也是其它项目的两倍


同理，项目的值是其它项目值的一半，那么占据的剩余空间也是其它项目的一半


（2）flex-shrink

flex-shrink属性指定了内部成员的收缩规则，当内部成员默认宽度之和大于容器宽度的时候就会自动进行收缩。默认值为1，表示空间不足，项目将进行收缩，设置值为0，表示项目不进行收缩。

如果设置的值不为0，且所有项目值相同，则所有项目均分收缩，即所有项目最终的宽度相同


此处盒子中的数字代表flex-shrink的值


如果设置的值不为0，且其中某一项目的值为2，其它项目的值为1，则值为2的项目缩放比例更大，即最终宽度比其它项目宽度更小


（3）flex-basis

flex-basis属性指定了内部成员在主轴方向上的初始大小，设置了flex-grow属性，那么容器内部成员的width属性是不生效的，如果需要指定某个内部成员的宽度，使用flex-basis属性即可。

如下将盒子1的flex-basis设置成300px，那么盒子1就占据了固定空间，剩下的空间分配给盒子2和盒子3。


（4）flex

flex属性是flex-grow、flex-shrink、flex-basis的简写属性

.item{
flex:flex-grow flex-shrink flex-basis
}

后面两个是可选，默认值是 flex:0 1 auto表示当有剩余空间时也不进行缩放，当空间不足时进行收缩，宽度不固定。

（5）align-self

align-self属性允许单个项目与其他项目使用不一样的对齐方式，可覆盖align-items属性，默认值是auto，表示继承父元素的align-items属性。

它的可取值和父容器的align-items可取值相同，并多了一个auto值

.container{
align-items:center;
}
.item{
align-self:flex-end;
}


（6）order

order属性规定了弹性容器中的成员的排列顺序，成员按照order属性值的增序进行排列，拥有相同order属性值的成员则按照标签出现的顺序进行排列。默认值为0，就是数值越小的元素，排列越靠前。

<style>
.item1{ order:1 }
.item2{ order:2 }
.item3{ order:0 }
</style>
<div class="item item1">1</div>
<div class="item item2">2</div>
<div class="item item3">3</div>
