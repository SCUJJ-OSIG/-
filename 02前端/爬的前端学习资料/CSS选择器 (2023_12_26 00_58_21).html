CSS选择器

来自： 青牛前端冲击营
万万
2023年02月23日 17:28

CSS选择器是用来指定网页上要进行样式设置的元素，它可以告诉浏览器那个HTML元素应当被选中应用CSS规则。被CSS选择器选中的元素叫做“选择器对象”。


  1. Id选择器：具有特定Id的元素 （#id）
  2. 类选择器：具有特定className的元素 （.class）
  3. 元素选择器：所有该类型的元素 （div）
  4. 属性选择器：拥有特定属性的元素 （input[type='text']）
  5. 伪类选择器：特定状态下的特定元素 （li:hover）
  6. 伪元素选择器：指定一个元素的某个部分 （p::first-line）
  7. 后代选择器：一个元素内指定的所有子孙元素 （div p）
  8. 子代选择器：一个元素内指定的第一级子元素 （div>p）
  9. 兄弟选择器：相邻的同级元素 （h1+p）
  10. 通配符选择器：匹配所有的元素（*）


Id选择器


id选择器是唯一的，在一个页面中只允许唯一id选择器存在

<div id="wrap"></div>
<style>
#wrap{}
</style>


类选择器


类选择器是HTML标签中 class属性指定的 className，允许多个相同的类选择器存在

<div class="list-item"></div>
<div class="list-item"></div>
<style>
.list-item{}
</style>


类型选择器


类型选择器也是元素选择器，指向所有指定类型的HTML元素

<div>
<p></p>
<p></p>
<p></p>
</div>

<style>
p{}
</style>


属性选择器


属性选择器是，指向指定元素指定标签属性的元素

<div id="box">
<input type="text">
<input type="text">
<input type="text">
</div>
<style>
input[type="text"]{
border:2px solid red;
height:30px;
}
</style>


伪类选择器


用来样式化一个元素的特定状态，如 :hover，指定鼠标悬浮在元素上的样式

<div class="box"></div>
<style>
.box{
width:30px;
height:30px;
background:#ccc;
}
.box:hover{
background:#c00000;
}
</style>


后代选择器


指向指定元素内包裹的指定元素，可以指向深层嵌套的元素

<div class="box">
<p>后代选择器的字体颜色</p>
<div class="info">
<p>深层嵌套的后代选择器的字体颜色</p>
</div>
</div>
<style>
.box p {
color:#c00000;
}
</style>


子代选择器


指向指定元素内包裹的第一层的指定元素，不对深层嵌套的元素起作用

<div class="box">
<p>后代选择器的字体颜色</p>
<div class="info">
<p>深层嵌套的后代选择器的字体颜色</p>
</div>
</div>
<style>
.box>p {
color:#c00000;
}
</style>


兄弟选择器


指向与指定元素相邻的同级元素

<div class="box"></div>
<div class="info"></div>
<div class="info"></div>
<style>
.box+.info {
background:#c00000;
width:30px;
height:30px;
}
</style>


通配符选择器


通配符选择器指向所有的元素，可以与其它的选择器配合使用

<style>
*{} // 代表匹配所有的元素
*[attr-dec="card"] { // 匹配所有具有属性 attr-dec 并且值属性值为card的元素
background:#c00000;
width:30px;
height:30px;
}
</style>



通配符选择器 < 类型选择器 < 类选择器 < Id选择器
